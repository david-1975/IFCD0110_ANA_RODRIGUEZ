Tablas

Etiquetas que definen una tabla

Una tabla HTML se define con la <table> etiqueta.

Cada fila de la tabla se define con la etiqueta <tr>. Un encabezado de la tabla se define con la etiqueta <th>. Por defecto, encabezados de la tabla están en negrita y centrado. A los datos / celda de la tabla se define con la etiqueta <td>.

Etiqueta Descripción
<table> Define una tabla
<tr> Define una fila
<th> Define el encabezado de la tabla
<td> Define una celda dentro de una fila
<caption> Define el título o leyenda de la tabla
<colgroup> Especifica un grupo de una o más columnas para darle formato
<col> Especifica propiedades de columna para cada elemento que contiene
<thead> Agrupa el contenido del encabezado
<tbody> Agrupa el contenido del cuerpo
<tfoot> Agrupa el contenido del pie

El atributo Scope

El atributo scope especifica si una celda de encabezado es una cabecera de una columna, fila o grupo de columnas o filas. No tiene ningún efecto visual en los navegadores web normales, pero puede ser utilizado por los lectores de pantalla.

El atributo Colspan y Rowspan

El atributo colspan define el número de columnas de una celda debe ocupar. El atributo rowspan especifica el número de filas de una celda debe ocupar.

El atributo Colgroup y Col

La etiqueta <colgroup> especifica un grupo de una o más columnas en una tabla para el formato. La etiqueta <colgroup> es útil para la aplicación de estilos para columnas completas en lugar de repetir los estilos para cada celda, para cada fila.

La etiqueta <col> especifica las propiedades de columna para cada columna dentro de un <colgroup>. La etiqueta <col> es útil para la aplicación de estilos para columnas completas, en lugar de repetir los estilos para cada celda, para cada fila.
