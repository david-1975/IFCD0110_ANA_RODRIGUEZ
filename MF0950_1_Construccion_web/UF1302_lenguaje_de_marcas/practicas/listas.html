Etiquetas que definen una Lista

Etiqueta Descripción
<ul> Define una lista desordenada
<ol> Define una lista ordenada
<li> Define los items de la lista
<dl> Define una lista de definición
<dt> Define un item de la lista de definición
<dd> Define un término de la lista de definición

Nota: Los elementos de lista pueden contener nueva lista, y otros elementos HTML, como imágenes y enlaces, etc.

Listas desordenadas (ul)

La etiqueta <ul> define una lista desordenada (viñetas). Utilice la etiqueta <ul> junto con <li> para crear listas desordenadas.

Atributo Valor Descripción

Listas ordenadas (ol)

La etiqueta <ol> define una lista ordenada. Una lista ordenada puede ser numérico o alfabético. Utilice <li> para definir elementos de la lista.

  1. Coffee
  2. Tea
  3. Milk
  1. Coffee
  2. Tea
  3. Milk
  1. Coffee
  2. Tea
  3. Milk

Listas de definición (dl)

La <dl> etiqueta define una lista de definición. La etiqueta <dl> se utiliza en conjunción con <dt> (define términos / nombres) y <dd> (describe cada término / nombre).

Coffee
Black hot drink
Vegetal origin
Milk
White cold drink
Animal origin
