¿Qué es multimedia?

Multimedia viene en muchos formatos diferentes y puede ser casi cualquier cosa que se pueda escuchar o ver (imágenes, música, sonido, vídeos, discos, películas, animaciones, etc.)

HTML video.

Antes de HTML5, un vídeo sólo se podía jugar en un navegador con un plug-in (como Flash). El HTML5 la etiqueta <video> especifica una forma estándar para incrustar un vídeo en una página web.
En HTML5, hay 3 formatos de vídeo: MP4, WebM, y Ogg.

Etiqueta y atributos Descripción
<video> Define el video
<source> Especifica recursos de medios múltiples para los elementos <picture>, <audio>, o <video>
<track> Define los subtítulos
autoplay El video comenzará a reproducirse automáticamente tan pronto como sea posible, sin detenerse para terminar de cargar los datos.
controls Permite que el usuario controle la reproducción de video, incluyendo volumen, búsqueda y pausar/reanudar reproducción.
heigh La altura del área de visualización del vídeo en píxeles
loop Al alcanzar el final del video, volverá automáticamente hasta el principio.
muted Especifica que el video está silenciado
poster Una URL que indica una imagen para mostrar hasta que el usuario reproduzca el video.
preload Especifica si el video va, o no, a cargarse al iniciar la página.
src La URL del vídeo que se va a insertar.
width La anchura del área de visualización del vídeo en píxeles

Ejemplo de uso

Big Buck Bunny

Código

  <video width="100%" height="auto" controls autoplay preload="auto" loop poster="./img/conejo_multimedia.jpg">
    <source src="video/mov_bbb.mp4" type="video/mp4">
    <source src="video/mov_bbb.ogg" type="video/ogg">
  </video>

HTML audio.

La etiqueta <audio> define sonido, como la música u otros flujos de audio. Actualmente, hay 3 formatos de archivo admitidos: MP3, WAV y OGG.
Se trata de una etiqueta nueva de HTML5

Etiqueta y atributos Descripción
<audio> Define el audio
<source> Especifica recursos de medios múltiples para los elementos <picture>, <audio>, o <video>
autoplay El audio comenzará a reproducirse automáticamente tan pronto como sea posible, sin detenerse para terminar de cargar los datos.
controls Permite que el usuario controle la reproducción de audio, incluyendo volumen, búsqueda y pausar/reanudar reproducción.
loop Al alcanzar el final del audio, volverá automáticamente hasta el principio.
muted Especifica que el audio está silenciado
preload Especifica si el audio va, o no, a cargarse al iniciar la página.
src La URL del audio que se va a insertar.

Ejemplo de uso

Audio

Código

  <audio controls>
    <source src="audio/intro.mp3" type="audio/mp3">
    <source src="./audio/intro.ogg" type="audio/ogg">
    <source src="./audio/intro.wav" type="audio/wav">
  <//audio>

Elemento Object

Define un objeto incrustado dentro de un documento HTML. Se utiliza para insertar plug-ins (como applets de Java, los lectores de PDF, Flash Players) en las páginas web.
Es soportado por todos los navegadores

Ejemplo de uso

Object flash

Código

<object width="400" height="50" data="./swf/bookmark.swf"></object>

Elemento iFrame

La etiqueta <iframe> especifica un marco en línea. Un marco en línea se utiliza para insertar otro documento dentro del documento HTML actual.
Para hacer frente a los navegadores que no soportan <iframe>, se añade un texto entre la etiqueta de apertura y el cierre.

Ejemplo de uso

Video Youtube

Código

<iframe width="560" height="315" src="https://www.youtube.com/embed/MEVEUTnFzu8" frameborder="0" allow="accelerometer; autoplay; encrypted-media; gyroscope; picture-in-picture" allowfullscreen></iframe>
