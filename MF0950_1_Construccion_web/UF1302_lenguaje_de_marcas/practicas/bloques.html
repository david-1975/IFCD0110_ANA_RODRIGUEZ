Etiquetas que citan el texto

Las abreviaturas de marcado <abbr> pueden proporcionar información útil a los navegadores, sistemas de traducción y motores de búsqueda. Utiliza el atributo title para definir el significado de la abreviación.

La etiqueta de marcado <cite> hace referencia a un trabajo creatico como un libro, una película, un programa de TV o una canción, no al nombre de una persona.

La etiqueta de marcado <blockquote> define una cita larga que e representa como un elemento de bloque. No genera automáticamente comillas alrededor del contenido.

La etiqueta de marcado <q> define una cita corta en línea. Marca las citas a otros autores o documentos. Genera automáticamente comillas alrededor del contenido.

La etiqueta de marcado <dfn> define un término y debe contener el atributo <title> para la definición/explicación del término que contiene.

Elementos que agrupan el texto

Elelemento <div>se utiliza como un contenedor para otros elementos HTML. Tiene atributos, aunque no son necesarios. Los más comunes son style, class e id para dar estilos CSS al bloque de contenido.

El elemento <span> se utiliza como un contenedor para agrupar un texto en línea. Esta etiquta también tiene atributos, aunque , como en el caso anterior, son innecesarios.
