El formato del texto

El formato de texto es el marcado que se aplica a un texto para añadirle datos de forma

Al teto resultante se le conoce como texto formateado, texto con estilos o texto enriquecido

Encabezados de texto

Los motores de búsqueda utilizan los encabezados para indexar la estructura y el contenido de sus páginas web.

Los usuarios hojear sus páginas por sus títulos. Es importante utilizar las partidas para mostrar la estructura del documento.

Los encabezamientos <h1>deben ser utilizados para las partidas principales, seguido por los encabezamientos <h2>,<h3>...que son menos importantes .

Nota Usar encabezados HTML para sólo los encabezados. No utilice encabezados para que el texto GRANDE o negrita.

Etiquetas que dan formato al texto

HTML también define especiales elementos para la definición de texto con un especial significado.

HTML utiliza elementos como <b>y <i>para el formato de salida, como negrita o cursiva texto.

Etiqueta Descripción Ejemplo Utilización
<b> Este texto está en negrita Este texto está en negrita <b>Este texto está en negrita</b>
<i> Este texto está en cursiva Este texto está en cursiva <i>Este texto está en cursiva</i>
<strong> Define un texto importante Este texto es importante <strong>Este texto es importante</strong>
<em> Este texto está en enfatizado Este texto está en enfatizado <em>Este texto está en enfatizado</em>
<small> Describe un texto más pequeño Define un texto más pequeño Ésto es un texto <small>más pequeño</small>
<sup> Describe texto en superíndice Describe texto en superíndice Describe texto en <sup>superíndice/sup>
<sub> Describe texto en suberíndice Describe texto en suberíndice Describe texto en <sub>subíndice/sub>
<ins> Describe texto insertado o agregado Describe texto insertado o agregado Describe texto insertado o <ins>agregado</ins>
<u> Describe texto subrayado Describe texto subrayado Describe texto <u>subrayado</u>
<s> Describe texto tachado Describe texto tachado Describe texto <s>tachado</s>
<del> Describe texto está tachado o eliminado Describe texto está tachado o eliminado Describe texto está <del>tachado</del>o eliminado
<mark> Describe texto marcado Describe texto marcado Describe texto <mark>marcado</mark>

Nota Los navegadores muestran <strong> como <b>, y <em>como <i>. Sin embargo, hay una diferencia en el significado de las siguientes etiquetas: <b> e <i>define el texto en negrita y cursiva respectivamente, pero <strong>y <em>significan que el texto es "importante".

Etiquetas que separan el texto

Las etiquetas <p> separan el texto, añadiendo automaticamente un espacio antes y despues de cada elemento <p>. El navegador eliminará todos los espacios en blanco y las lineas adicionales cuando se visualiza la etiqueta.

La etiqueta <br> es una etiqueta vacia, lo que significa que no tiene ninguna etiqueta final. Inserta un salto de línea sin comenzar un nuevo párrafo.

Un texto dentro de una etiqueta <pre> se muestra en una fuente de ancho fijo (preferiblemente Courrier) y conserva ambos espacios y saltos de linea
