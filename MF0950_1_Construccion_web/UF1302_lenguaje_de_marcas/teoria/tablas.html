HTML Tablas

Etiquetas que definen una tabla

Una tabla HTML se define con la <table>etiqueta.

Cada fila de la tabla se define con la <tr> etiqueta. Un encabezado de la tabla se define con la <th> etiqueta. Por defecto, encabezados de la tabla están en negrita y centrado. A los datos / celda de la tabla se define con la <td> etiqueta.

Etiqueta Descripción
<table> Define la estructura de una tabla
<tr> Define una fila en una tabla
<th> Define una celda de cabecera en una tabla
<td> Define una celda dentro de una fila en una tabla
<caption> Define el título o leyenda de una tabla
<colgroup> Especifica en un tabla un grupo de una o más columnas para darle formato
<col> Especifica propiedades de columna para cada columna dentro de un elemento <colgroup>
<thead> Agrupa el contenido del encabezado en una tabla
<tbody> Agrupa el contenido del cuerpo en una tabla
<tfoot> Agrupa el contenido del pie de página en una tabla

El atributo Scope

El atributo scope provee un mecanismo para indicar explícitamente a qué celdas afecta una celda de encabezado. El atributo scope no tiene ningún efecto visual en los navegadores web normales, pero puede ser utilizado por los lectores de pantalla.

El atributo colspan y rowspan

El atributo colspan define el número de columnas de una celda debe ocupar. El atributo rowspan especifica el número de filas de una célula debe ocupar.

El atributo colgroup y col

La etiqueta <colgroup> especifica un grupo de una o más columnas en una tabla para el formato. La etiqueta <colgroup> es útil para la aplicación de estilos para columnas completas, en lugar de repetir los estilos para cada célula, para cada fila.

La etiqueta <col> especifica las propiedades de columna para cada columna dentro de un <colgroup> elemento. La etiqueta <col> es útil para la aplicación de estilos para columnas completas, en lugar de repetir los estilos para cada célula, para cada fila.
